# React Lab

## What's it?
This's a Design System project. This was built with React, Storybook, Tailwind and Radix.

## Main concepts:
Components composition, tokens and utility classes.


## Requirements:
1. `node 16.x or 10.x`

## Running the project (locally):

1. `cd react-lab`
2. `yarn`
3. `yarn storybook`
